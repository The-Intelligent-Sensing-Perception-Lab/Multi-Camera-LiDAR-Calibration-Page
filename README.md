# Joint Multi-Camera LiDAR Extrinsic Calibration — Project Website

This repository hosts the **project website** for our paper:

*Joint Multi-Camera LiDAR Extrinsic Calibration via Learned Pairwise Initialization and Geometric Refinement*

Presented at the **URVIS Workshop** ([Unified Robotic Vision with Cross-Modal Sensing and Alignment](https://urvis-workshop.github.io/)), **CVPR 2026**, Denver.

Source code for the calibration method will be released in a separate repository when available.

## Authors

Aziz Al-Najjar, Marzieh Amini, James R. Green, Felix Kwamena — Carleton University, Ottawa, Canada
